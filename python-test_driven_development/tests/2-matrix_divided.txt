Exemples
--------

.. code-block:: python

    >>> matrix = [[1, 2], [3, 4]]
    >>> matrix_divided(matrix, 2)
    [[0.5, 1.0], [1.5, 2.0]]

    >>> matrix_divided([[10, 20], [30, 40]], 10)
    [[1.0, 2.0], [3.0, 4.0]]

    >>> matrix_divided([[5.5, 3.3], [7.7, 8.8]], 1.1)
    [[5.0, 3.0], [7.0, 8.0]]
